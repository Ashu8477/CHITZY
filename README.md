# 💬 Chitzy – Modern Chat Website

Chitzy is a **modern, responsive chat-style website** designed to provide a clean UI and smooth user experience. The project focuses on **frontend fundamentals, UI/UX design, and interactive components**, making it a strong portfolio and interview-ready project.

## 📌 Project Overview

Chitzy is built to simulate a **chat / messaging platform interface**, showcasing:

- Clean and modern UI design
- Responsive layouts for all devices
- Interactive components and animations
- Scalable frontend structure

This project demonstrates my ability to build **real-world web interfaces** using modern web technologies.

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/your-username/chitzy.git
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
